# 检查申请管理


## 分页查询检查申请列表


**接口地址**:`/api/v1/check-request`


**请求方式**:`GET`


**请求数据类型**:`application/x-www-form-urlencoded`


**响应数据类型**:`*/*`


**接口描述**:<p>支持按挂号ID、检查状态和创建时间范围筛选</p>



**请求参数**:


| 参数名称 | 参数说明 | 请求类型    | 是否必须 | 数据类型 | schema |
| -------- | -------- | ----- | -------- | -------- | ------ |
|pageNum|页码，默认1|query|false|integer(int32)||
|pageSize|每页条数，默认10|query|false|integer(int32)||
|registerId|挂号ID|query|false|integer(int32)||
|checkState|检查状态|query|false|string||
|creationTimeStart|创建时间起始（含），格式yyyy-MM-dd HH:mm:ss|query|false|string(date-time)||
|creationTimeEnd|创建时间截止（含），格式yyyy-MM-dd HH:mm:ss|query|false|string(date-time)||


**响应状态**:


| 状态码 | 说明 | schema |
| -------- | -------- | ----- | 
|200|OK|ResultPageResultCheckRequestResponse|


**响应参数**:


| 参数名称 | 参数说明 | 类型 | schema |
| -------- | -------- | ----- |----- | 
|code|状态码|integer(int32)|integer(int32)|
|message|提示信息|string||
|data||PageResultCheckRequestResponse|PageResultCheckRequestResponse|
|&emsp;&emsp;total|总记录数|integer(int64)||
|&emsp;&emsp;pageNum|当前页码|integer(int32)||
|&emsp;&emsp;pageSize|每页条数|integer(int32)||
|&emsp;&emsp;records|当前页数据列表|array|CheckRequestResponse|
|&emsp;&emsp;&emsp;&emsp;id|检查申请ID|integer(int32)||
|&emsp;&emsp;&emsp;&emsp;registerId|挂号ID|integer(int32)||
|&emsp;&emsp;&emsp;&emsp;medicalTechnologyId|医技项目ID|integer(int32)||
|&emsp;&emsp;&emsp;&emsp;techName|医技项目名称(冗余)|string||
|&emsp;&emsp;&emsp;&emsp;checkInfo|检查信息|string||
|&emsp;&emsp;&emsp;&emsp;checkPosition|检查部位|string||
|&emsp;&emsp;&emsp;&emsp;creationTime|创建时间|string(date-time)||
|&emsp;&emsp;&emsp;&emsp;checkEmployeeId|检查员工ID|integer(int32)||
|&emsp;&emsp;&emsp;&emsp;checkTime|检查时间|string(date-time)||
